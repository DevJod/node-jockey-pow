Jogo Jockey Pow para Node.js
===

Jogando...

    node jockey-pow.js pedra
    player: pedra
    cpu: tesoura
    vencedor: pedra

Instalando...

    git clone https://github.com/devfuria/node-jockey-pow


Rodando os testes...

    cd /node-jockey-pow
    npm install
    npm test
